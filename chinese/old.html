Public Commands  (just say em in the channel)
join - Joins the game

whats the damage - Displays lifetime winnings

Setting Hands  (/notice these commands to Annette_15)
set a b c d e - Sets the back hand with cards a,b,c,d,e
set f g h i j - Sets middle hand after front hand is set. Automatically sets remaining 3 cards to front
unset - Unsets all hands and returns to prompt for back hand
natural (flush/straight/pairs/dragon) - After setting, use this command if you have a natural (otherwise it won't count!)
done - Confirms set hands, once done, no changes can be made
surrender - Surrenders your hand
Useful aliases use em!
/set /notice annette_15 set $$1 $2 $3 $4 $5
/d /notice annette_15 done
/unset /notice annette_15 unset
/nat /notice annette_15 natural $$1
/surr /notice annette_15 surrender

Playing for Clams  say em in the channel *NYI*
!challenge [player] X - Challenges a player currently sitting to play for $X a point
!accept [player] - Accepts a challenge from the indicated player
!deny [player] - Denies a challenge from the indicated player
!end [player] - Ends an ongoing challenge with the indicated player
!stakes - Displays your ongoing challenges

Payouts & Bonus Hands
