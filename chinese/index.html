Public Commands  (just say em in the channel)
chinese poker time - Starts the bot up
join - Joins the game. If the game is full, joins the wait list
quit - Quits the game. If currently in a hand, removes you at the end of the hand
start - Starts the game
whats the damage - Displays lifetime winnings
status - Displays the current players. If a hand is in progress, displays time since deal and which players are done setting
clock - Calls the clock, only available 90 seconds after the deal. Once called, hand will end in 30s, unless...
time - Requests time after the clock has been called, adding another 60 seconds
NOTE: Timed out hands will be treated as a surrender

Setting Hands  (/notice these commands to Annette_15)
set a b c d e - Sets the back hand with cards a,b,c,d,e
set f g h i j - Sets middle hand after back hand is set. Automatically sets remaining 3 cards to front
unset - Unsets all hands and returns to prompt for back hand
natural (flush/straight/pairs/dragon) - After setting, use this command if you have a natural (otherwise it won't count!)
done - Confirms set hands, once done, no changes can be made
surrender - Surrenders your hand
Useful aliases use em!
/set /notice annette_15 set $$1 $2 $3 $4 $5
/d /notice annette_15 done
/unset /notice annette_15 unset
/nat /notice annette_15 natural $$1
/surr /notice annette_15 surrender

Playing for Clams  say em in the channel
stakes [player] [X] - Challenges a player currently sitting to play for $X a point (Respond with [y/n])
stakes - Displays all ongoing challenges
To end a challenge, simply set stakes to 0, no confirmation required

Payouts & Bonus Hands
Channel commands *NYI*
!change payout x y z - Initiates a vote to change the payout: +x for winning 2 hands, +y for a sweep, -z for surrender
!change bonus [name] - Initiates a vote to change the bonus structure
!vote [y/n] - Votes on the current proposal, majority needed to change
Notice commands *NYI*
bonus - Displays current bonus values
listbonus [name] - Displays bonus values for bonus set [name]. If no name is indicated, displays a list of the bonus sets
